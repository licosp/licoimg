# パッケージコーディング規約

## 目的
コードベースの技術基準とプロジェクト固有のルールを確立します。

## 技術スタック
- **フレームワーク**: Vite + Vanilla JavaScript
- **スタイリング**: モダンCSS（変数、Flexbox/Grid）
- **デプロイ**: GitHub Pages（`docs/` ディレクトリ）

## 規約
- **ソース**: `src/`（エントリーポイント: `src/main.js`）
- **デザイン**: プレミアムでモダンな美学（グラスモーフィズム、ダークモード）。
- **アーキテクチャ**: クライアントサイドのみ（サーバーサイドロジックなし）。

## ルール
- **プレースホルダーなし**: アセットには `generate_image` を使用し、プレースホルダーのテキストや画像は使用しません。
- **SEO**: タイトル、メタディスクリプション、セマンティックHTMLを含めます。
