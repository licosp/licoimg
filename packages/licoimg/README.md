## ğŸš€ Getting Started

### Prerequisites

- **Node.js** (v14 or higher)
- **npm** or **yarn**

### Installation & Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/licosp/licoproj.git
   cd licoproj/packages/licoimg
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   
   Open your browser at the URL displayed in the terminal (usually `http://localhost:5173`)

4. **Build for production**
   ```bash
   npm run build
   ```
   
   Production files will be generated in the `docs/` directory
